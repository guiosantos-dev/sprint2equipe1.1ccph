⚡ ChargeGrid Intelligence

📌 Visão Geral do Projeto

O **ChargeGrid Intelligence** é uma prova de conceito desenvolvida para o GoodWe Challenge, com o objetivo de simular um sistema inteligente de gerenciamento de carregamento de veículos elétricos em ambientes comerciais.

O sistema demonstra como a infraestrutura de recarga pode ser otimizada através de:

* Controle inteligente de demanda energética
* Tarifação automática de consumo
* Interoperabilidade entre diferentes carregadores
* Uso de lógica baseada em inteligência artificial para redistribuição de energia


🧠 Contexto do Problema

Com a expansão dos veículos elétricos, surgem desafios importantes como:

* Sobrecarga da rede elétrica em horários de pico
* Falta de padronização entre equipamentos
* Dificuldade de cobrança e monetização do serviço
* Necessidade de gestão inteligente de múltiplos carregadores simultâneos

O ChargeGrid Intelligence foi criado para resolver esses problemas de forma simulada e didática.


🚀 Evolução do Projeto

✔ Sprint 1

Na primeira etapa, foi realizada a análise teórica do problema, incluindo:

* Estudo da empresa GoodWe
* Definição do conceito de ChargeGrid
* Análise dos pilares:

  * Controle de demanda
  * Protocolos abertos

✔ Sprint 2 (Atual)

Nesta etapa, foi desenvolvida uma **prova de conceito funcional**, contendo:

* Sistema funcional em Python
* Interface gráfica com Tkinter
* Simulação de carregadores elétricos

* Redistribuição automática de energia (IA simulada)
* Cálculo de tarifação e receita
* Simulação de interoperabilidade entre diferentes marcas


⚙️ Funcionalidades

🔋 Controle de Demanda

O sistema monitora o consumo total de energia e evita sobrecarga da rede elétrica através da redistribuição automática de potência.


🤖 Inteligência Artificial (Simulada)

Uma lógica automatizada identifica quando há excesso de consumo e redistribui energia entre os carregadores para estabilizar o sistema.


💰 Tarifação e Pagamento

O sistema calcula automaticamente o valor da recarga com base no consumo energético:

* Fórmula: `kWh consumido × tarifa`
* Tarifa simulada: R$ 1,20 por kWh


🔌 Interoperabilidade

O sistema simula carregadores de diferentes fabricantes operando em conjunto:

* GoodWe
* ABB
* WEG
* Carregador adicional (veículo conectado)


🧱 Arquitetura do Sistema

```
Carregadores → Monitoramento → Cálculo de Consumo → IA de Controle → Redistribuição de Energia → Tarifação
```

🖥️ Tecnologias Utilizadas

* Python 3
* Tkinter (Interface Gráfica)
* Programação estruturada
* Lógica de simulação de sistemas inteligentes


▶️ Como Executar o Projeto

1. Instale o Python 3.x
2. Baixe o repositório
3. Execute o arquivo:

```bash
python main.py
```

4. Utilize os botões da interface:

* Atualizar sistema
* Conectar veículo
* Ativar IA
* Resetar sistema


📊 Demonstração do Sistema

O sistema simula três estados principais:

1. Estado normal

Consumo dentro do limite da rede.

2. Sobrecarga

Quando o consumo ultrapassa o limite definido.

3. Correção automática

A IA redistribui energia e estabiliza o sistema.
